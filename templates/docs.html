Documentation

Complete guide to setting up and using your Intelligent Farm OS.

1. Connecting Your Hardware

EcoMonitor Pro relies on serial communication to ingest live sensor data. Ensure your Arduino or ESP32 is flashed with the correct sketch that outputs comma-separated JSON data over serial at 9600 baud.

// Expected Serial Output Format from Arduino:
{"temperature": 24.5, "humidity": 60.2, "soil": 450, "rain": "No", "pump": "OFF"}
  • Plug your microcontroller into your computer via USB.
  • Navigate to the Dashboard.
  • In the left sidebar, click the refresh icon next to the COM port dropdown.
  • Select your designated COM port and click Connect.

2. Configuring the Gemini AI API

The "AI Agronomist" chat feature relies on Google's Gemini 2.5 Flash model. You must provide your own API key to activate this feature.

  • Create a file named .env in the root directory of the application.
  • Open the file and add your API key exactly as shown below:
GEMINI_API_KEY="AIzaSyYourSecretKeyHere..."

If you hit rate limits (Free Tier restrictions), the AI chat will notify you. Wait a few minutes and try again.

3. Using the AI Agronomist

Once connected, the AI Agronomist constantly monitors your live data stream in the background.

  • Click the floating spark button in the bottom right corner of the dashboard to open the chat window.
  • Every message you send automatically includes the current sensor state, so the AI always knows exactly what's happening on your farm right now.
  • Use the quick-action "Chips" above the chat input to instantly ask common questions like "Analyze trends" or "Why is the pump on?".

4. Exporting Data

All telemetry data is saved locally to a SQLite database (sensor_data.db). You can easily export this data for external analysis.

  • Go to the Dashboard.
  • Click the Export Data CSV button in the left sidebar under Data Management.
  • A complete CSV file containing every timestamped reading will automatically download to your computer.
